## Краткое описание файлов:

### Основные файлы:
- **`main.py`** - Запуск приложения, инициализация БД и GUI
- **`database.py`** - Класс Database для работы с PostgreSQL/SQLite
- **`config.py`** - Конфигурация приложения, загрузка из .env
- **`.env`** - Конфиденциальные данные (пароли, настройки БД)
- **`requirements.txt`** - Список зависимостей Python
- **`library.db`** - SQLite база данных (альтернативная версия)

### Графический интерфейс:
- **`gui/auth_window.py`** - Окно авторизации с проверкой логина и пароля
- **`gui/main_window.py`** - Главное окно с навигацией и дашбордом
- **`gui/books_window.py`** - Управление книгами (CRUD операции, поиск)
- **`gui/loans_window.py`** - Управление выдачами книг, контроль сроков
- **`gui/users_window.py`** - Управление пользователями и правами доступа
- **`gui/reports_window.py`** - Генерация отчетов, экспорт в PDF

### Статические ресурсы:
- **`static/styles.css`** - Файл стилей для кастомизации интерфейса

## Установка и запуск:
```bash
# Установка зависимостей
pip install -r requirements.txt

# Настройка окружения
cp .env.example .env  # и настройте параметры подключения к БД

# Запуск приложения
python main.py
```
